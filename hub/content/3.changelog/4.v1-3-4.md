**新增功能：**

- **一键安装器** - Docker Compose 全生命周期管理，安装/升级/备份/卸载一键搞定
- **FlecHub 门户中心** - 项目官方门户，包含文档、主题、更新日志等
- **博客主题模板** - 完整的主题模板系统，支持二次开发
- **FlecPanel 管理后台** - 轻量级管理面板，版本管理与公告发布

**优化改进：**

- **Node.js 升级** - 基础镜像升级到 22 版本
- **依赖更新** - @nuxtjs/seo 升级到 5.1.3
- **性能优化** - ECharts 按需引入，Mermaid 懒加载

**问题修复：**

- 修复邮件通知发件人格式处理
- 修复代码块复制功能异常
- 更新一言 API 接口地址

**发布地址：** <https://github.com/talen8/FlecBlog/releases/tag/v1.3.4>
